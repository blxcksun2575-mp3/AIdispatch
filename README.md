    🤖 Aimylogic Scripts

    Учебные сценарии чат-ботов на платформе Aimylogic.

    > 🚧 In progress — проект в активной разработке (июль 2026).
    > Создан в рамках подготовки к вакансии AI-Dispatcher Developer.

    📚 Что внутри

    - autoservice-bot/ — запись клиента в автосервис (слоты: марка, проблема, бюджет, время)
    - clinic-bot/ — запись к врачу
    - fallback-template/ — шаблон 3-уровневой fallback-иерархии
    - escalation-template/ — передача оператору

    🛠 Стек

    - Платформа: Aimylogic (Just AI)
    - Языки: RU, KZ (в перспективе)
    - Каналы: Aimychat, Telegram
